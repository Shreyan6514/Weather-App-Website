### 1. Clone the repository:
```bash
git clone https://github.com/Shreyan6514/Weather-App-Website.git
```

### 2. Navigate into the project folder:
```bash
cd Weather-App-Website
```

### 3. Install dependencies:
```bash

```

### 4. Add your environment variables:

